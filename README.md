# AI扫盲课 - 8课深化版

AI扫盲课 - 用大白话讲清楚人工智能的8个核心理论

## 📚 课程大纲（新8课）

| 课次 | 主题 | 核心内容 |
|:---:|:---|:---|
| **1** | 概率的世界 | 概率建模、Softmax、温度参数 |
| **2** | 注意力 | Self-Attention、Transformer架构 |
| **3** | Tokenizer | BPE算法、词切分、子词机制 |
| **4** | 提示词工程 | Prompt设计、Few-shot、思维链(CoT) |
| **5** | 幻觉与局限 | AI为什么会胡说、识别与防范 |
| **6** | 涌现能力 | Scaling Law、量变引发质变 |
| **7** | 对齐理论 | RLHF、PPO、价值观对齐 |
| **8** | 多模态基础 | 文本+图像融合、CLIP、跨模态理解 |

## 🎯 课程逻辑

```
基础层（1-2课）：AI如何"思考"
    ↓
输入层（3课）：文本如何处理
    ↓
应用层（4-5课）：如何使用 + 问题识别
    ↓
高级层（6-7课）：能力涌现 + 安全对齐
    ↓
拓展层（8课）：多模态前沿
```

## 🚀 使用方式

1. 安装 VS Code + Marp for VS Code 插件
2. 打开任意课程文件
3. 点击预览按钮查看幻灯片
4. 可导出为 PDF / PPTX / HTML
